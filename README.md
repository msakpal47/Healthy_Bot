# Healthy Bot

## Project Structure
- `app.py`: Main entry point for the Flask application.
- `core/`: Contains core logic.
  - `gen_ai.py`: LLM and PDF handling.
  - `consult.py`: Medical consultation logic.
- `static/`: CSS and JavaScript files.
- `templates/`: HTML templates.
- `Data/`: Data files (CSV, etc.).
- `storage/`: Generated PDFs and logs.

## Running the App
Run the application using:
```powershell
python app.py
```
